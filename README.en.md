# EntryCodeQR

#### Description
2020年新冠病毒肆虐，百姓恐慌。全国一级戒备，各地小区纷纷限制进出。
此小程序是基于小程序的二维码电子通行证，旨在建立无接触式可回溯的门岗通行记录。通行记录同步计入“链飞”区块链，感谢链飞科技的无私扶持。
